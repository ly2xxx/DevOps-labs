# DevOps Labs

Hands-on labs and guides for DevOps tools and practices.

## 📂 Contents

### AWS CloudFormation & Service Catalog Quick Start
**Location:** `aws-quickstart/`

A comprehensive 1-hour tutorial covering:
- ✅ CloudFormation basics (Infrastructure as Code)
- ✅ Multi-resource stack deployments
- ✅ Service Catalog self-service setup
- ✅ 3 hands-on labs with free tier resources

**Start here:** [aws-quickstart/START-HERE.md](aws-quickstart/START-HERE.md)

**What you'll learn:**
- Define AWS infrastructure as code (YAML templates)
- Deploy entire environments with one command
- Provide self-service infrastructure to teams
- Implement governance and compliance controls

**Time:** ~45 minutes (plus 10 min setup)  
**Cost:** Free tier eligible (S3, EC2, RDS)

### GCP Serverless Quick Start
**Location:** `gcp-quickstart/`

A practical 1-hour tutorial covering:
- ✅ Cloud Run containerized deployments
- ✅ Cloud Scheduler cron jobs
- ✅ Cloud Logging and monitoring
- ✅ Real project: Streamlit health checker

**Start here:** [gcp-quickstart/START-HERE.md](gcp-quickstart/START-HERE.md)

**What you'll build:**
- Deploy containerized apps to Cloud Run
- Schedule tasks with Cloud Scheduler
- Monitor and log with Cloud Logging
- Replace local cron jobs with cloud solution

**Time:** ~1 hour (includes real project deployment)  
**Cost:** $0/month (free tier + $300 credit)

**Real-world project:**
- Streamlit health checker that wakes sleeping apps
- Fully containerized and serverless
- Replaces local Windows cron jobs
- Production-ready monitoring
