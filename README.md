### AWS CloudFormation & Service Catalog Quick Start
**Location:** `aws-quickstart/`

A comprehensive 1-hour tutorial covering:
- ✅ CloudFormation basics (Infrastructure as Code)
- ✅ Multi-resource stack deployments
- ✅ Service Catalog self-service setup
- ✅ 3 hands-on labs with free tier resources

**Start here:** [aws-quickstart/START-HERE.md](aws-quickstart/START-HERE.md)

**What you'll learn:**
- Define AWS infrastructure as code (YAML templates)
- Deploy entire environments with one command
- Provide self-service infrastructure to teams
- Implement governance and compliance controls

**Time:** ~45 minutes (plus 10 min setup)  
**Cost:** Free tier eligible (S3, EC2, RDS)
